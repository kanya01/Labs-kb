## Scope

Issues covering the `/discover` page implementation:
- Unified discover page layout and routing
- Role-based filter (Producer / Vocalist / Engineer)
- Experience level filter
- Keyword search across name, username, bio, skills
- User card component with portfolio preview and inline audio playback
- Backend API endpoint for filtered user discovery
- Pagination / load-more behaviour
